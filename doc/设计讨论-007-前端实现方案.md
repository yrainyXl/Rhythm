## 一、技术实现总览

### 核心组件架构

```
入口层 (EntryLayer)
├── 背景层 (AtmosphereBackground)        — 时段色温 + 状态纹理
├── 内容层 (MomentContent)               — 五种此刻的内容
│   ├── 行动时刻 (ActionMoment)
│   ├── 停靠时刻 (RestMoment)
│   ├── 念头时刻 (ThoughtMoment)
│   ├── 收束时刻 (ClosureMoment)
│   └── 空白时刻 (SilentMoment)
├── 交互层 (GestureLayer)                — 长按/左右滑/轻点/停顿/下滑
└── 过渡层 (TransitionController)        — 入口层到骨架层的下沉动画
```

### 状态判断逻辑（入口层打开时）

```
1. 获取当前时间 → 确定时段色温（清晨/白天/夜晚）
2. 获取用户数据 →
   - 是否有进行中实践？
   - 今天是否已记录？
   - 是否有刚结束的实践等待复盘？
   - 近期（7天内）是否有重复出现的主题/关键词？
3. 优先级判定 →
   - 有未记录实践 + 白天 → 行动时刻
   - 有未记录实践 + 晚间 → 停靠时刻（或行动时刻，看上一次打开时间）
   - 实践刚结束 → 收束时刻
   - 无实践 + 有重复主题 → 念头时刻
   - 什么都没有 → 空白时刻
```

### 状态机

```
       ┌──────────┐
       │ 空白时刻  │ ← 默认 / 无任何待处理
       └────┬─────┘
            │ 用户发起方向/实践引导
            ▼
       ┌──────────┐
       │ 行动时刻  │ ← 有进行中实践 + 未记录
       └────┬─────┘
            │ 今日已记录 / 晚上
            ▼
       ┌──────────┐
       │ 停靠时刻  │ ← 晚上 + 未记录（允许什么都不做）
       └────┬─────┘
            │
            ▼
       ┌──────────┐
       │ 念头时刻  │ ← 无实践 + 有重复主题
       └────┬─────┘
            │ 实践周期结束
            ▼
       ┌──────────┐
       │ 收束时刻  │ ← 实践结束 + 等待复盘
       └──────────┘
```

## 二、五种时刻的前端形态

### 1. 行动时刻

**背景：** 清晨冷色调 + 轻微的呼吸光晕
**布局（从上到下）：**
- 极淡的日期 + 星期（opacity: 0.4, font-size: 13px）
- 中央呼吸圆（直径约 120px，CSS animation: breathe 4s ease-in-out infinite）
- 圆下方：实践名称（font-size: 18px, font-weight: 500）
- 再下方：第 X 天（14px, opacity: 0.6）
- 最小行动描述（14px, opacity: 0.5）

**动画：**
- 圆呼吸：scale(1) ↔ scale(1.05)，周期 4s
- 长按：圆的边框从虚线变实线，3 秒填满
- 填满瞬间：圆变完成色（柔和绿），短暂放大 → 回弹 → 稳定
- 完成文字淡入：「这 3 分钟，已经留在今天里。」（fadeIn 1s）
- 随后轻问文字淡入：「更接近平静了，还是只是完成了？」（fadeIn 0.5s）

**交互：**
- 长按圆 → 完成记录（3 秒填满）
- 右滑圆 → 跳过（圆飘走，文字「没关系，今天还是你的」）
- 左滑圆 → 快速感受（三个表情散开）
- 向下滑动 → 过渡到骨架层

### 2. 停靠时刻

**背景：** 暖橙色调 + 极其缓慢的渐变
**布局：**
- 中央上方：柔和的时间显示（22:46，大字号但低对比度）
- 中间一行文字：「今天不需要补齐什么。」（16px）
- 下方：「愿意留下一点关于今天的感受吗？」（14px, opacity: 0.5）
- 中央呼吸圆（比行动时刻的圆更小、更透）

**交互：**
- 长按圆 → 弹出极简输入区（无边框 textarea，支持语音）
- 不操作直接离开 → 不追问
- 停留超过 10 秒不做操作 → 文字缓慢变换：「也可以什么都不留。晚安。」（自动消退）

### 3. 念头时刻

**背景：** 中性色调 + 极淡的文字纹理背景（像纸张上的水印）
**布局：**
- 上方：浮现的关键词/句子（来自用户近期的记录）
  - 字体略大，用衬线或无衬线但更有温度的字重
  - 如果有多条，第二条更淡、第三条几近透明
- 中间：「它好像还没有离开。」（15px）
- 下方两个轻选项，不是按钮，是两行淡色文字：
  - 「想再想想」（opacity: 0.7，hover 时变亮）
  - 「先放在这里」（opacity: 0.4）

**交互：**
- 轻点「想再想想」→ 过渡到输入的花园，定位到相关认知
- 轻点「先放在这里」→ 文字沉淀消失，入口层消退
- 下滑 → 进入骨架层

### 4. 收束时刻

**背景：** 过渡色调 + 缓慢收拢的光效
**布局：**
- 上方：实践名称 + 完成天数
- 中间：「这段练习，已经陪了你 7 天。」
- 下方：「它有没有让你靠近一点，原本想成为的状态？」
- 中央圆微微发光（比行动时刻更亮）

**复盘三问（每问一屏）：**
- 第 1 问：「开始时，你希望什么变得不一样？」— 自由输入 + 安静等待
- 第 2 问：「这几天，真实发生了什么？」— 自由输入
- 第 3 问：「你想留一句什么给以后的自己？」— 自由输入
- 第 4 问：「下一步你想怎么做？」— 三个轻选项：一直保持 / 再试一轮 / 先放一放

每问之间用缓动过渡（当前问淡出 → 下一问淡入，约 0.8s）

**结束：** 「记住了。这条实践已经加入你的方法地图。」→ 画面下沉到回声房间

### 5. 空白时刻

**背景：** 极简，几乎没有纹理，纯粹的色温底
**布局：**
- 中央一行字：「此刻，没有什么需要完成。」（16px, opacity: 0.6）
- 下方：「想留下什么时，这里都在。」（14px, opacity: 0.3）
- 一个极小的圆（直径约 40px，非常淡）

**交互：**
- 长按小圆 → 留下一个念头（自由输入）
- 不做任何操作 → 保持静止
- 下滑 → 进入骨架层

## 三、过渡动画

### 入口层 → 骨架层（下滑过渡）
- 整个入口层画面向上滑动并逐渐变淡
- 骨架层从下方滑入
- 过渡时长约 400ms，缓出函数 cubic-bezier(0.4, 0, 0.2, 1)

### 入口层内部状态切换
- 不同时刻之间不自动切换，只在用户下次打开时重新判定
- 复盘对话内的问题切换用 fadeIn/fadeOut（约 800ms）

## 四、实现提案

### 技术选型
- 动画：Framer Motion（手势 + 布局动画）
- 背景色温：CSS 自定义属性 + Tailwind 暗色模式变体
- 圆呼吸：CSS @keyframes + SVG filter
- 语音输入（停靠时刻）：Web Speech API（MediaRecorder + 前端转文字或直接存储音频）

### 关键文件规划
```
src/
├── features/
│   └── entry-layer/
│       ├── EntryLayer.tsx              — 主组件，状态路由
│       ├── moments/
│       │   ├── ActionMoment.tsx        — 行动时刻
│       │   ├── RestMoment.tsx          — 停靠时刻
│       │   ├── ThoughtMoment.tsx       — 念头时刻
│       │   ├── ClosureMoment.tsx       — 收束时刻
│       │   └── SilentMoment.tsx        — 空白时刻
│       ├── components/
│       │   ├── BreathingCircle.tsx     — 呼吸圆组件
│       │   ├── AtmosphereBackground.tsx— 时段色温背景
│       │   └── GestureLayer.tsx        — 手势处理
│       └── store/
│           └── entry-store.ts          — 入口层状态管理
```

### 数据需求
- 当前用户的进行中实践（含第几天、最小行动、是否已记录）
- 刚结束等待复盘的实践
- 近期（7 天）用户的认知卡片/记录关键词提取
- 上一次打开 App 的时间
- 当前本地时间
